Step 1: Save Your Photo

Save your profile photo to your computer. Make sure it's a high-quality image.

Step 2: Rename the File

Rename your photo file to: profile-outdoor.jpg

Step 3: Copy to Portfolio Folder

Copy the renamed photo to this folder:

D:\Downloads\cursor\assets\images\

Replace the existing placeholder file.

Step 4: View Your Portfolio

Open your index.html file in your browser. Your photo will automatically appear in the About section!

Alternative Method (Drag & Drop)

You can also:

  1. Open File Explorer
  2. Navigate to D:\Downloads\cursor\assets\images\
  3. Drag and drop your photo file directly into this folder
  4. Rename it to profile-outdoor.jpg when prompted

Your portfolio is already configured to use this image - once you add the file, it will appear automatically!
